# NestJS 核心概念速查

## CLI 命令

| 命令 | 作用 |
|------|------|
| `nest -h` | 查看所有指令 |
| `nest new nest-basic` | 创建项目 |
| `nest g mo auth` | 生成模块 |
| `nest g s auth` | 生成服务 |
| `nest g c auth` | 生成控制器 |

## 装饰器速查

| 装饰器 | 作用 |
|--------|------|
| `@Controller` | 定义控制器路由 |
| `@Get` / `@Post` / `@Put` | HTTP 方法 |
| `@Param` | 路由参数 |
| `@Body` | 请求体 |
| `@Injectable` | 可注入服务 |
| `@Module` | 模块定义 |
| `@Catch` | 异常捕获 |

## 六大机制

| 机制 | 实现接口 | 装饰器 | 作用 |
|------|---------|--------|------|
| Middleware | NestMiddleware | @Injectable | 洋葱模型，请求/响应处理 |
| Guards | CanActivate | @Injectable | 身份校验 |
| Pipes | PipeTransform | @Injectable | 数据转换 |
| Interceptors | NestInterceptor | @Injectable | 响应拦截 |
| Filters | ExceptionFilter | @Catch | 异常处理 |

## Module 结构

```
@Module({
  imports: [],      // 导入其他模块
  controllers: [],  // 控制器
  providers: [],    // 服务白名单
  exports: [],      // 对外暴露
})
```

## 跨模块服务共享

```
providers 注册 → exports 导出 → 其他模块 imports 导入
```
